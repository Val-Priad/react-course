## 🧠 What I Learned

I completed this React course during the summer after my second year of university, before starting my third year. The course gave me a much deeper understanding of modern frontend development and helped me build a stronger technical foundation.

One of the most valuable parts of the course was that learning React also improved my understanding of other technologies. While studying new concepts, I started to better understand things I had previously skipped or only partially understood. The more I learned about React, JavaScript, rendering, state management, and application architecture, the more connections I started to see between different areas of web development.

One of the most interesting parts of the course was learning how different mechanisms work under the hood. I really enjoyed discovering how the V8 JavaScript engine works, how React's Virtual DOM is used, and how it interacts with the real DOM to make UI updates more efficient. I also learned how to work with state correctly, how React re-renders components, and how more advanced tools like useReducer can help manage more complex state logic in a cleaner and more predictable way.

This course had a major impact on my third year of university. React made many academic projects much easier to build, especially tasks related to web applications, user interfaces, state management, routing, reusable components, and API integration. Since then, I have used React almost every month for different university assignments and personal learning projects.

Overall, this course gave me practical skills and fundamental knowledge that I continue to use regularly. It helped me become more confident in building frontend applications and made web development feel much more structured and understandable.

### ⚛️ React Fundamentals

- Practiced creating reusable React components
- Improved understanding of JSX syntax and component structure
- Learned how to pass data between components using props
- Practiced rendering lists with `map()`
- Used conditional rendering to control UI output

### 🧩 State Management

- Practiced managing local component state with `useState`
- Used callback functions when updating state based on previous state
- Learned how to lift state up between related components
- Practiced working with derived state
- Used `useReducer` for more complex state logic

### 🔄 Effects and Browser APIs

- Practiced fetching data with `useEffect`
- Used `AbortController` to cancel outdated API requests
- Worked with `localStorage` for persistent data
- Practiced custom hooks for reusable logic
- Used keyboard events to improve user interaction

### 🎨 UI and UX

- Built interactive UI components such as accordions, quizzes, lists, and rating widgets
- Practiced showing loading and error states
- Improved form handling and controlled inputs
- Practiced user-friendly conditional messages
